# Splatoon 3 - Project Thunder
The biggest Splatoon 3 modpack, featuring a port of every map from Splatoon 1 & 2.

## We've done a lot..
The pack will include:
- All Splatoon 1-2 Maps
- Custom Scenes, seperated so you can still play S3 maps
- Music ported from Splatoon 1 & 2
- Night Stages
- Accurate/Updated Envs, Skyboxes
- Accurate/Updated Layouts, Modes
