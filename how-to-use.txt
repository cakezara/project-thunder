# How to setup Project Thunder

-- On Switch, and Ryujinx

1. Drag the 'atmosphere' folder to the root of your SD Card
2. Make sure you're on version 11.0.0/11.0.1, which are the versions Project Thunder supports.
3. We recommend starting with a full save file, but it's okay if not.
